# BÁO CÁO CÁC MODULE CHỨC NĂNG - DỰ ÁN EZTRAVEL (PRJ301)

## 1. Module Tour & Khám Phá (Client)
- **Tính năng**: Xem danh sách tour, lọc theo loại, tìm kiếm, so sánh tour.
- **Trạng thái**: 100% Hoàn thành.
- **Output**: Trang chủ (`home.jsp`), Trang khám phá (`explore.jsp`).

## 2. Module Đặt Tour & Thanh Toán (Booking)
- **Tính năng**: Giỏ hàng trọn gói, tính toán giá theo mùa, tích hợp QR SePay Banking, Voucher/Coupon.
- **Trạng thái**: 100% Hoàn thành.
- **Output**: Trang thanh toán (`checkout.jsp`), Lịch sử đơn hàng.

## 3. Module Tư Vấn (Consultation) - MỚI
- **Tính năng**: Form gửi yêu cầu tư vấn tour riêng biệt tích hợp Google Maps, gửi trực tiếp về Admin.
- **Trạng thái**: 100% Hoàn thành.
- **Output**: Landing page consultation section.

## 4. Module Admin Dashboard (Quản trị)
- **Tính năng**: Quản lý Tour (CRUD), Quản lý khách hàng, Quản lý yêu cầu tư vấn, Phân quyền người dùng.
- **Trạng thái**: 100% Hoàn thành.
- **Output**: Hệ thống trang `/admin/*`.

## 5. Module AI & Analytics (Đặc sắc)
- **Tính năng**: Chatbot hỗ trợ du lịch bằng Llama 3, Dashboard phân tích doanh thu bằng AI.
- **Trạng thái**: 100% Hoàn thành (Điểm cộng).
- **Output**: Nút Chatbot góc màn hình, Trang Dashboard AI.
